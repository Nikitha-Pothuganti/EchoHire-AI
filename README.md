# 🎙️ EchoHire AI
**Humanizing the First Round of Hiring with AI-Driven Voice Interviews.**

## 🚀 Overview
EchoHire AI is a voice-first interview assistant designed to screen candidates efficiently while maintaining a human touch. It uses **Gemini 3 Flash** for intelligence and **Murf AI Falcon** for realistic voice interaction.

## 🛠️ Tech Stack
- **Frontend:** Streamlit
- **Brain:** Google Gemini 3 Flash
- **Voice:** Murf AI (Falcon Model)
- **Ear:** SpeechRecognition (Google Web API)

## 🏗️ Installation
1. Clone the repo: `git clone https://github.com/Nikitha-Pothuganti/EchoHire-AI.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file with your `GOOGLE_API_KEY` and `MURF_API_KEY`.
4. Run the app: `streamlit run app.py`
